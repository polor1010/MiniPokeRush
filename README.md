####  暴擊和閃躲的規則

| 規則    | 觸發時機      | 機率                | 效果說明                                   |
|---------|---------------|---------------------|---------------------------------------------|
| 閃躲    | 每次攻擊前    | 10%（一般）         | 攻擊完全無效，被攻擊方顯示「躲過了攻擊！」   |
|         |               | 30%（屬性被剋制時） |                                             |
| 暴擊    | 每次攻擊命中時| 10%                 | 傷害加倍（x2），訊息顯示紅色「暴擊!」        |
